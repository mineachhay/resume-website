# Minea Chhay Resume Website

Personal resume website for Minea Chhay, built as a single-page portfolio with a downloadable PDF resume.

## Live Hosting

This project is prepared for GitHub Pages hosting at:

https://mineachhay.github.io/resume-website/

If the repository name or Pages URL changes, update these values in [index.html](./index.html):
- `canonical`
- `og:url`
- `og:image`
- `twitter:image`

## Local Files

- `index.html`: main resume website
- `resume-pdf.html`: print-friendly resume source
- `cv-minea-chhay.pdf`: downloadable PDF resume
- `profile.jpg`: profile and social preview image
- `style.css`: main styling
- `script.js`: site interactions
